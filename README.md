# Fall Detection Module – Smart Companion for Alzheimer’s Patients

This project implements a fall detection system using an ESP32 and MPU6050 sensor.
A machine learning model is trained to classify fall and non-fall events using accelerometer and gyroscope data.

## Current Features
- ESP32 sensor data acquisition
- Machine learning model training
- Local live fall prediction using Python

## Technologies
- ESP32
- MPU6050
- Python (scikit-learn)
- Arduino IDE
